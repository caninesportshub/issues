## Tips for Great Issues
- Use a clear, descriptive **title** (e.g., "Timer freezes during low-network Flyball race on Android tablet")
- Include as much detail as possible:
  - What you were doing
  - What happened (and what you expected)
  - Device / OS / Browser
  - Screenshots or short videos (drag & drop them right into the issue!)
  - App version (check in settings/about)
- If reporting a bug, steps to reproduce are super helpful
- **Do NOT** include sensitive information (passwords, personal details, payment info, etc.)
